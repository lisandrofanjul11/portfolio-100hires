B2B SaaS Research: LinkedIn Organic Content Strategy
This repository contains a high-signal research project on LinkedIn organic content strategies for B2B SaaS.

Experts Selected
I selected 10 practitioners who actively run B2B SaaS GTM motions. These voices were chosen for their focus on tactical frameworks rather than generic advice (Positioning, Buyer Psychology, GTM Strategy).

Repository Structure
/research/sources.md: List of experts and rationales.

/research/linkedin-posts/: Tactical frameworks extracted from recent high-signal posts.

Why this focus?
LinkedIn is currently the primary GTM engine for B2B SaaS, and I focused on distilling actionable frameworks that can be immediately applied to content operations.
